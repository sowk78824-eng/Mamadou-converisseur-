# Road Trip Planner - Convertisseur de Vitesse

## Description
Application web permettant de convertir des distances (KM/MI) et de calculer le temps de trajet selon la vitesse moyenne.

## Fonctionnalités
- Conversion bidirectionnelle Kilomètres ⇄ Miles.
- Trois vitesses prédéfinies : Ville (50), Route (80), Autoroute (130).
- Gestion des erreurs (entrées non numériques ou vides).

## Installation
Il suffit d'ouvrir le fichier `index.html` dans un navigateur web.
